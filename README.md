# Mutual Fund Analytics Platform

Bluestock Fintech Capstone Project

## Objective

Build a complete Mutual Fund Analytics Platform using:

- Python
- Pandas
- SQLite
- Power BI
- AMFI Data
- mfapi.in

## Datasets

1. Fund Master
2. NAV History
3. AUM Data
4. SIP Inflows
5. Category Inflows
6. Folio Counts
7. Scheme Performance
8. Investor Transactions
9. Portfolio Holdings
10. Benchmark Indices
